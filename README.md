# Volatility-Adaptive Trading Bot with Web Dashboard

This repository contains a Python-based trading bot for Gold (`XAUUSD`) and Silver (`XAGUSD`) connecting to Exness via MetaTrader 5. 

The bot automatically switches strategies dynamically depending on the market regime:
*   **Ranging Regime (Low Volatility):** Mean-reversion trading using Bollinger Bands and RSI on the M5 timeframe, exiting at the middle band.
*   **Trending Regime (High Volatility):** Momentum breakout trading using 9/21 EMA crossovers on the M5 timeframe with dynamic ATR-based Stop Loss and Take Profit levels.

It includes a beautiful, premium dark-mode web dashboard optimized for mobile phones (using viewport limits, touch target sizes, and card transformations).

---

## 🛠️ Features

*   **Regime Switching:** Measures 14-period ATR on the M15 chart against its 50-period SMA to determine high vs. low volatility regimes.
*   **Dynamic Lot Sizing:** Automatically calculates lot sizes based on your chosen Risk Profile and the distance to the Stop Loss.
*   **Risk Profiles:** Switch between **Safe (1%)**, **Moderate (2.5%)**, and **Risk (5%)** risk settings directly from the dashboard.
*   **Environment Configuration:** Runs seamlessly in **Mock Simulator Mode** on Linux or Windows when testing, and connects to the **Real MT5 Terminal** on Windows when configured.
*   **Mobile-First Design:** Collapses wide tables into individual cards automatically on screen widths below `768px` for easy phone monitoring.

## 💻 Setup and Installation

### 1. Install Dependencies
Ensure you have Python 3.10+ installed. Install the required libraries:
```bash
pip install fastapi uvicorn pandas numpy websockets
```
*Note: If you are running on Windows to trade with real data, install the MetaTrader 5 library as well:*
```bash
pip install MetaTrader5
```

### 2. Configure Environment Variables
Create a file named `.env` in the root directory (or modify the template):
```env
# Set to False on Windows to connect to MT5 and trade with Exness
# Set to True on Linux/Windows for testing and running with simulated mockup data
DRY_RUN=True

# Optional: Specify custom file path to terminal64.exe if it's in a non-default location
# MT5_PATH=C:\Program Files\Exness MetaTrader 5\terminal64.exe
MT5_PATH=
```

---

## 🚀 Running the Application

### 1. Start the Server
Run the FastAPI backend using Uvicorn:
```bash
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```
Open your browser and navigate to: `http://localhost:8000`.

### 2. Connect from your Mobile Phone
To monitor or control the bot on your phone, tunnel your local port to the internet using **Ngrok**:
```bash
ngrok http 8000
```
Copy the secure `https://...` link generated by Ngrok and load it on your phone.
